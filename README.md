# UWB Software Library for FreeRTOS
The library provides ranging and communication capabilities using the DW1000 UWB module from Decawave. Out of the box, it is meant to be used with the open-source hardware/software drone platform from Bitcraze (Crazyflie) and the Loco Positioning Deck, but it can be easily adapted to work on any FreeRTOS based system.

The code will be uploaded after the paper acceptance.
